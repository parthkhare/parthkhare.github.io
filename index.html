SociocaRtography R · Machine Learning · GIS

Parth Khare

I am Data Scientist, currently managing Data Science divison at Nielsen India. I have worked extensively on R, Decision Tree based machine learning modules and Satellite imagery data. Recently, I successfully developed a semi-automated model architecture 'GRID' in R. The GRID is one of first ever solutions to report socio-economic attributes at a 1 square kilometer precision level . The algorithm works by interacting Public Census data with Micro Satellite Imagery data to unearth spatially intelligent patterns.

I have developed several customized functions to make geo-processing in R easier In addition I have worked creating optimsiation function for Gradient Boosting Model to help in focused identifcation. of local and global minima. Also, since R has so many ways of doing the same analysis, I have over time created a repository to help in direct programming based challenges faced in R R Utility Kit.
